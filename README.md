# Webots Line Follower Robot

Проект робота, движущегося по линии в симуляторе Webots. Робот умеет уверенно следовать по трассе, проходить перекрестки, заходить в тупиковые ответвления с контрольными точками (КТ) и автоматически возвращаться на главную дорогу для продолжения пути.

## Особенности и алгоритм навигации
- **Line Tracking**: Надежное следование по линии благодаря 3 инфракрасным датчикам земли (ground sensors).
- **Распознавание перекрестков**: Робот детектирует ответвления, используя боковые датчики, и останавливается для принятия решения.
- **Относительная память поворотов**: При встрече с перекрестком робот отдает приоритет левым, затем правым поворотам. При попадании в тупик (или на КТ) робот выполняет разворот на 180 градусов, возвращается на перекресток и, используя память о последнем маневре, выезжает обратно на основную трассу.
- **Остановка на финише**: Автоматическое определение координат финишной точки и полная остановка работы при ее достижении.

## Структура проекта
- `line-follower-robot-w/line_follower_robot/controllers/lfr_py/lfr_py.py` — главный Python-скрипт контроллера, содержащий логику управления моторами и навигацией.
- `line-follower-robot-w/line_follower_robot/worlds/` — файл мира Webots и изображение трассы (`track.bmp`).
- `requirements.txt` — зависимости проекта.

## Установка и запуск
1. Установите симулятор [Webots](https://cyberbotics.com/).
2. Склонируйте данный репозиторий на свой компьютер.
3. Откройте файл мира (например, `.empty_world.wbproj` / `empty_world.wbt`) в Webots.
4. Нажмите кнопку **Play** в интерфейсе Webots, чтобы запустить симуляцию. Скрипт контроллера `lfr_py.py` подцепится и запустится автоматически.

## Зависимости
Проект работает "из коробки" и использует встроенный API Webots (`from controller import Robot`). Дополнительная установка внешних пакетов через `pip` не требуется.
